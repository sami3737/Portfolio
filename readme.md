## Projet principal : Rust Evolution

Rust Evolution est une plateforme que j’ai conçue et développée permettant de proposer des plugins et fonctionnalités pour serveurs Rust.

### Objectifs du projet

* Créer un écosystème de plugins personnalisés
* Proposer une boutique en ligne fonctionnelle
* Automatiser la gestion des fonctionnalités serveur

---

## Stack technique

* **Frontend** : HTML, CSS, JavaScript
* **Backend** : PHP
* **Base de données** : MySQL
* **CMS / outils** : WordPress (customisation avancée)
* **Serveur** : gestion de serveurs de jeu Rust (uMod / Oxide)

---

## Fonctionnalités développées

* Système de boutique (plugins téléchargeables)
* Développement et intégration de plugins Rust
* Gestion des données utilisateurs et produits
* Intégration Discord / serveur de jeu
* Automatisation de certaines actions serveur

---

## Challenges techniques

* Optimisation des performances côté serveur
* Gestion des interactions entre plugins
* Structuration des données utilisateurs
* Intégration entre site web et serveur de jeu
